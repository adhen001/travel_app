## ⚙️ Fitur Utama

- Login sederhana (belum terhubung ke backend)
- Tampilan utama dengan background dan daftar destinasi
- Tombol booking (Hotel, Pesawat, Kegiatan, Bus, Kereta)
- Ulasan destinasi populer dengan rating
